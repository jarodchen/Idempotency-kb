# 幂等性设计知识库

系统化、结构化的幂等性设计技术知识库，基于 VitePress 构建。

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

本地访问: http://localhost:5173/idempotency-kb/

## 📖 知识体系

- **基础概念** - 幂等性的定义、HTTP方法、安全性
- **核心实现模式** - Token机制、唯一索引、乐观锁、分布式锁
- **技术栈实现指南** - ASP.NET Core、PostgreSQL、MySQL、Redis
- **场景化解决方案** - 支付交易、订单系统、库存扣减
- **架构与运维** - API网关、监控告警、压力测试
- **常见问题与陷阱** - 性能优化、一致性保证、边界情况
